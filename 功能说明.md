# 功能说明 - 当前无 API 多线程版

## 当前版本功能

### 1. 删除 API 下载

本版不再调用 API 下载，不再请求：

```text
music.douyin.com/api/...
douyin.com/aweme/v1/web/music/detail/...
```

只解析分享页面和页面源码里的可用音频地址。

### 2. 多线程剪贴板监听

选择菜单 `2` 或运行：

```bat
py main.py --clipboard
```

复制一个汽水音乐分享链接，程序会自动加入下载队列。上一首还没下载完，也可以继续复制下一首。

### 3. 不弹白色空白页面

Selenium/Chrome 默认无头运行：

```python
HEADLESS = True
```

同时隐藏 ChromeDriver 控制台窗口，减少弹窗干扰。

### 4. 60秒以内短音频自动删除

下载完成后先检测原始文件时长，转换后再检测最终文件时长。

```text
≤ 60 秒：自动删除
> 60 秒：保留
```

用于过滤疑似试听片段。

### 5. 智能链接提取

支持完整分享文本：

```text
《歌曲名》@汽水音乐 https://qishui.douyin.com/s/xxxx/
```

也支持纯链接。

### 6. MP4 自动转 MP3

默认配置：

```python
AUTO_CONVERT = True
OUTPUT_FORMAT = "mp3"
BITRATE = "320k"
KEEP_ORIGINAL = False
```
