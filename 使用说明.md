# 汽水音乐爬虫使用说明 📖

## 重要提示 ⚠️

### 版权限制
汽水音乐的歌曲有版权保护机制。如果遇到以下情况，说明该歌曲无法下载：

- 提示"该歌曲没有版权或已下架"
- 提示"未找到音频下载链接"
- 页面显示"很抱歉，目前暂不支持播放该歌曲"

**解决方法：** 尝试其他有版权的歌曲链接。

### 如何获取有效链接

1. 打开汽水音乐APP
2. 找到一首**可以正常播放**的歌曲
3. 点击分享按钮
4. 复制分享链接
5. 将链接粘贴到本程序中

## 快速开始 🚀

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行程序
```bash
python main.py
```

### 3. 选择操作
- 输入 `1` - 下载新歌曲
- 输入 `2` - 转换已下载的MP4文件

### 4. 输入链接

**支持两种输入方式：**

✅ **方式1: 直接粘贴完整分享文本（推荐）**
```
《One Of The Girls x After Hours》@汽水音乐 https://qishui.douyin.com/s/iQfNnBfJ/
```
程序会自动识别并提取链接，无需手动删除其他文字！

✅ **方式2: 只粘贴链接**
```
https://qishui.douyin.com/s/iQfNnBfJ/
```

**示例输出：**
```
✓ 识别到链接: https://qishui.douyin.com/s/iQfNnBfJ/
  歌曲: One Of The Girls x After Hours
```

## 常见问题 ❓

### Q1: 为什么有些歌曲下载不了？
**A:** 该歌曲可能：
- 没有版权授权
- 仅限APP内播放
- 已经下架

**解决：** 换一首可以正常播放的歌曲。

### Q2: 如何判断歌曲是否可下载？
**A:** 在汽水音乐APP中：
- 能正常播放 = 可能可以下载
- 显示"暂不支持播放" = 无法下载

### Q3: 程序提示"Selenium初始化失败"？
**A:** 确保：
1. Chrome浏览器已安装
2. selenium版本 >= 4.6
3. 关闭所有Chrome窗口后重试

```bash
pip install -U selenium
```

### Q4: 下载速度很慢？
**A:** 可能原因：
- 网络连接问题
- 服务器限速

**优化：** 检查网络连接，耐心等待。

### Q5: 转换MP3失败？
**A:** 需要安装ffmpeg：

**Windows:**
1. 下载ffmpeg: https://ffmpeg.org/download.html
2. 解压到某个目录
3. 将bin目录添加到系统PATH

**macOS:**
```bash
brew install ffmpeg
```

**Linux:**
```bash
sudo apt install ffmpeg
```
