# 使用说明

## 一、最简单用法

1. 解压压缩包。
2. 双击 `01_安装依赖.bat`。
3. 双击 `02_启动剪贴板监听.bat`。
4. 打开汽水音乐，复制分享链接。
5. 程序自动识别剪贴板并下载。

## 二、复制什么内容

可以复制完整分享文本：

```text
《歌曲名》@汽水音乐 https://qishui.douyin.com/s/xxxx/
```

也可以只复制链接：

```text
https://qishui.douyin.com/s/xxxx/
```

程序会自动提取有效链接。

## 三、下载模式

### 推荐：剪贴板自动下载

运行：

```bat
py main.py --clipboard
```

或双击：

```text
02_启动剪贴板监听.bat
```

特点：

- 复制链接后自动下载。
- 下载时还能继续复制下一首。
- 新链接会自动排队。
- 重复链接会自动跳过。

### 手动连续下载

运行：

```bat
py main.py --manual
```

或双击：

```text
04_手动连续下载.bat
```

每次下载完会直接提示输入下一条地址。

## 四、60秒以内自动删除

当前版本默认：

```text
≤ 60 秒：自动删除
> 60 秒：保留并转换
```

配置文件：

```text
src/config.py
```

相关参数：

```python
SHORT_AUDIO_THRESHOLD_SECONDS = 60
DELETE_SHORT_AUDIO = True
```

## 五、不弹白色空白页面

当前默认无头运行：

```python
HEADLESS = True
HIDE_BROWSER_WINDOW = True
```

如果你想调试，看浏览器页面，把它们改成：

```python
HEADLESS = False
HIDE_BROWSER_WINDOW = False
```

## 六、安装依赖失败怎么办

如果提示：

```text
'pip' 不是内部或外部命令
```

不要用 `pip`，用：

```bat
py -m pip install -r requirements.txt
```

如果 `py` 也不行，重新安装 Python，安装时勾选：

```text
Add python.exe to PATH
```

## 七、下载目录

文件保存在：

```text
downloads
```

如果开启自动转换，最终一般是 MP3。
